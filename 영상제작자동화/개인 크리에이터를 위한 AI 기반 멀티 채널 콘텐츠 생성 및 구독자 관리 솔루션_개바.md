네, 문서 데이터를 모두 확인했습니다. 이를 바탕으로 개발 지침 문서를 생성해 드리겠습니다.

# 개인 크리에이터를 위한 AI 기반 멀티 채널 콘텐츠 생성 및 구독자 관리 솔루션

## 1. 개요

-   **한 줄 정의**: 개인 크리에이터가 보유한 텍스트 기반의 지식 자산(예: 책 PDF)을 활용하여, AI를 통해 다양한 소셜 미디어 플랫폼용 콘텐츠를 자동으로 생성하고 구독자와 직접 소통할 수 있도록 돕는 종합 솔루션.
-   **배경 및 문제 인식**: 다수의 개인 크리에이터는 유튜브, 블로그, 인스타그램 등 여러 채널을 운영하지만, 각 플랫폼에 최적화된 콘텐츠를 꾸준히 생산하고 배포하는 데 많은 시간과 노력을 소모합니다. 특히 방대한 텍스트 형태의 지식을 보유한 전문가들은 이를 디지털 콘텐츠로 전환하고 확산하는 데 큰 어려움을 겪고 있으며, 이는 채널 성장 정체와 구독자 관리의 비효율로 이어집니다.
-   **이번 버전(릴리즈)의 목표**: 크리에이터가 기존 저작물(PDF)을 손쉽게 다채널 콘텐츠로 재가공하고, 이를 통해 확보된 구독자 정보를 기반으로 직접적이고 개인화된 소통을 수행하여 채널 영향력을 극대화하는 핵심 기능을 구현합니다.

## 2. 타깃 유저 & 사용 시나리오

-   **주요 유저 그룹 정의**:
    -   **관리자 (크리에이터)**: 1인 작가, 강사, 전문가 등 다량의 텍스트 기반 콘텐츠를 보유하고 있으나, 이를 소셜 미디어용으로 가공하고 배포하는 데 어려움을 느끼는 크리에이터. (예: 140권의 책을 출간한 작가)
    -   **일반 사용자 (구독자)**: 크리에이터의 콘텐츠를 소비하고, 뉴스레터나 알림 등을 통해 최신 소식을 받아보길 원하는 팬 또는 잠재 고객.
-   **대표 사용 시나리오**:
    1.  **콘텐츠 업로드 및 분석**: 작가(크리에이터)가 자신의 신간 도서 PDF 파일을 플랫폼에 업로드합니다.
    2.  **AI 콘텐츠 생성**: AI가 PDF 내용을 분석하여 책 소개 블로그 포스트, 인스타그램용 핵심 문구와 이미지, 숏폼 영상 스크립트, 뉴스레터 초안 등 다양한 형식의 콘텐츠를 자동으로 생성합니다.
    3.  **검토 및 수정**: 작가는 생성된 콘텐츠 초안들을 검토하고, 플랫폼의 편집 툴을 사용해 자신의 스타일에 맞게 수정합니다.
    4.  **콘텐츠 배포**: 수정이 완료된 블로그 글은 네이버 블로그에 예약 발행하고, 인스타그램 콘텐츠는 즉시 업로드합니다.
    5.  **구독자 소통**: 사전에 정보 제공에 동의한 구독자들에게 신간 소식과 콘텐츠 링크가 담긴 뉴스레터와 카카오톡 알림톡을 발송하여 구매 및 채널 참여를 유도합니다.

## 3. 핵심 가치 제안

-   **유저 문제 요약**: 크리에이터는 보유한 지식 자산을 여러 플랫폼에 맞게 가공하고 배포하는 데 과도한 시간을 소모하며, 구독자와의 직접적인 소통 채널 부재로 관계 구축에 어려움을 겪습니다.
-   **해결 방식 요약**: 원본 텍스트(PDF)만 업로드하면 AI가 각 채널에 최적화된 콘텐츠(블로그, SNS, 영상 스크립트 등)를 자동으로 생성해 배포 과정을 간소화하고, 구독자 정보를 직접 수집/관리하여 맞춤형 소통(뉴스레터, 알림톡 등)을 가능하게 합니다.
-   **차별점 / 기존 대안 대비 강점**:
    -   **텍스트 자산 특화**: 영상 요약 위주의 경쟁 서비스와 달리, 책과 같은 방대한 텍스트 콘텐츠의 심층 분석 및 재가공에 특화되어 있습니다.
    -   **종합 관리 솔루션**: 콘텐츠 생성을 넘어 구독자 정보 수집, 관리, 직접 소통(알림 발송)까지 올인원(All-in-one)으로 제공하여 플랫폼 종속성을 탈피하고 크리에이터 자산을 구축합니다.

## 4. 범위 (Scope)

-   **In Scope 기능**:
    -   구독자 정보 수집 및 알림 동의 기능 (랜딩페이지 기반)
    -   PDF 콘텐츠 업로드, AI 분석 및 이미지 생성 기능
    -   AI 기반 다중 플랫폼 콘텐츠 자동 생성 (블로그, 인스타그램, 롱폼/숏폼 영상, 뉴스레터) 및 하이라이트 선택 기능
    -   생성된 콘텐츠 검토 및 편집 기능
    -   콘텐츠 자동 배포 및 예약 기능 (유튜브, 인스타그램, 네이버 블로그/카페 연동)
    -   구독자 대상 알림(뉴스레터, 알림톡, SMS) 발송 및 참여 유도 시스템
    -   연동 플랫폼 계정 정보 통합 대시보드
-   **Out of Scope**:
    -   AI가 생성한 영상 스크립트를 기반으로 한 자동 영상 제작 기능
    -   실시간 구독자 채팅 및 커뮤니티 기능
    -   상세한 A/B 테스팅 기능
    -   콘텐츠 유료 판매 및 결제 시스템

## 6. UX 플로우 개요

-   **주요 플로우 다이어그램**:
    1.  **구독자**: `랜딩페이지 방문` -> `정보(이름, 이메일 등) 입력` -> `알림 수신 동의` -> `정보 제출` -> `완료`
    2.  **크리에이터 (콘텐츠 생성 및 배포)**:
        `로그인` -> `대시보드` -> `PDF 업로드` -> `AI 분석 및 하이라이트 제시` -> `하이라이트 선택` -> `AI 콘텐츠 생성(블로그, SNS 등)` -> `콘텐츠 검토 및 수정` -> `배포 채널 선택` -> `예약 또는 즉시 발행` -> `발행 완료`
    3.  **크리에이터 (알림 발송)**:
        `로그인` -> `구독자 관리 메뉴` -> `발송 대상 그룹 선택` -> `알림 메시지 작성 (생성된 콘텐츠 링크 포함)` -> `발송 채널(이메일, 알림톡) 선택` -> `발송` -> `결과 통계 확인`

-   **화면별 주요 인터랙션**:
    -   **랜딩 페이지**: 사용자가 스크롤 없이 핵심 가치를 인지하고, 바로 자신의 정보를 입력하여 구독 신청을 완료할 수 있도록 구성.
    -   **관리자 대시보드**: 연동된 채널 현황, 최근 생성된 콘텐츠 목록, 구독자 수 변화 등 핵심 지표를 위젯 형태로 시각화. '새 콘텐츠 만들기' 버튼으로 플로우 시작.
    -   **콘텐츠 생성 페이지**: 3단계(업로드 -> AI 분석/선택 -> 결과 확인)로 프로세스를 명확히 구분. 생성된 각 플랫폼별 콘텐츠를 탭(Tab) 형태로 제공하여 쉽게 전환하며 검토 및 수정.
    -   **콘텐츠 에디터**: 직관적인 WYSIWYG 에디터를 제공. 이미지 변경/업로드, 텍스트 서식 변경 등 기본 기능 지원.
    -   **구독자 관리 페이지**: 필터링 및 검색 기능을 통해 특정 구독자 그룹을 쉽게 찾고, 선택하여 알림 발송 대상으로 지정.

## 7. 비기능 요구사항

-   **성능**:
    -   PDF 업로드 및 AI 분석 시간: 100MB 미만 PDF 파일 기준, 2분 이내에 초기 분석 및 하이라이트 추출 완료.
    -   페이지 로드 시간: 주요 페이지(대시보드, 에디터)는 3초 이내에 로드 완료.
-   **보안**:
    -   구독자 개인정보는 암호화하여 저장하고, 개인정보보호법 등 관련 법규를 철저히 준수.
    -   외부 플랫폼 API 연동 시 OAuth 2.0 등 표준 인증 방식을 사용하고, 인증 토큰은 안전하게 관리.
-   **안정성**:
    -   외부 API(카카오톡, 네이버 등) 오류 발생 시, 시스템 내에 로그를 기록하고 관리자에게 실패 알림을 전송.
    -   예약 발행 기능은 서버 시간 기준으로 오차 범위 1분 이내에서 정확하게 동작해야 함.
-   **확장성**:
    -   향후 새로운 소셜 미디어 플랫폼(예: 틱톡, 링크드인)을 추가할 수 있도록 연동 모듈을 추상화하여 설계.
    -   콘텐츠 생성 AI 모델을 교체하거나 업데이트할 수 있는 유연한 구조 설계.
