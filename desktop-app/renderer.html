AutoForm Naver RPA

클라이언트 PC에서 네이버 블로그 업로드 자동화를 수행하는 로컬 에이전트입니다. 재부팅 후에도 로그인된 사용자 세션에서 자동 실행되도록 설계되어 있습니다.

런타임 상태

로컬 RPA 서버 확인 중
Chromium 번들 확인 중
네이버 세션 파일 확인 중
포트 -
앱 버전 -

네이버 로그인

Chromium 창이 열리면 직접 로그인하세요. 네이버 메인 또는 블로그 화면으로 이동하면 세션이 자동 저장됩니다. 세션 쿠키가 만료되면 이 버튼으로 다시 로그인하면 됩니다.

앱 설정

Windows 시작 시 자동 실행
사용자 로그인 직후 앱을 자동 실행합니다.

창을 닫아도 앱은 트레이에서 계속 실행됩니다.
